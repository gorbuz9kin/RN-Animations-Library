This is a React-Native Animations Library

# Install dependencies

## Step: 1

```bash
npm i
```

## Step: 2

```bash
cd ios/ && pod install && cd ..
```

# Getting Started

## Step 1: Start metro

```bash
npm run start
```

## Step 2: Start your Application

### For Android

```bash
npm run android
```

### For iOS

```bash
npm run ios
```

# Components list

- Spinning wheel
- Text animation
- Movies slider
